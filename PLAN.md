## Стадия 5: Голосование — фаза выбора
**Цель:** Инлайн-кнопки в группе, таймер 2 мин, авто-подсчёт, исключение.

**Механика:**
- `job_queue.run_once` с конфигурируемым таймером
- Досрочное завершение при голосе всех живых
- Удаление кнопок после подсчёта (`edit_message_reply_markup`)
- `mark_alibi_revealed()` для защиты от двойной публикации

**Чек-лист проверки:**
- [ ] В группе появляются кнопки с именами активных игроков (по 3–4 в ряд)
- [ ] Нажатие кнопки — голос учтён, можно сменить
- [ ] Через N секунд — авто-подсчёт
- [ ] Если все проголосовали досрочно — досрочный подсчёт
- [ ] Подсчёт: обычные голоса + штрафные (penalty_votes)
- [ ] Проверка явки >50%
- [ ] Ничья → возврат в discussion
- [ ] Явка <50% → возврат в discussion
- [ ] Исключение → публикация начального козыря → деактивация
- [ ] Если alive > 3 → возврат в discussion (кнопки в ЛС)
- [ ] Если alive ≤ 3 → уведомление: «Ведущий, используйте /final»
- [ ] `/use` работает во время голосования (+2 штрафных)
- [ ] Кнопки голосования удаляются после завершения
